# ops-alerts

Alertmanagerconfig for å sende teamalerts til Ops.

## Legg til nytt team

Når man skal legge til, eller fjerne et team, så må man redigere filen `alertmanagerconfig.yaml`.
Dette kan gjøres direkte i Github.

1. Åpne filen i Github.
2. Trykk på blyanten (edit this file).
3. Teams er definert på linje 51. Teams skilles med `|`.
4. Trykk på _Commit changes..._, og beskriv endringen din, og så trykk på _Commit changes_.
5. Så rulles det automatisk ut.
